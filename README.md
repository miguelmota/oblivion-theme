Oblivion Theme
==============

A Tumblr Theme 
--------------

#### Features
* Customizable colors
* Adjustable post width
* Disqus comments
* Display latest tweet
* Display likes
* Display following
* Like button
* Reblog button
* Share buttons
* Fixed navigation
* Endless scroll
* Masonry

Latest release: v1.2.6
